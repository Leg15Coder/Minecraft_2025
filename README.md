# Minecraft_2025 — клиентская сборка

Репозиторий содержит клиентскую сборку Minecraft под версию 1.21.1 (Fabric 0.18.4 + моды + текстуры), ипользуя кастомный клиент.

## Требования

- Установленный TLauncher (версия не старее 2.935) на ПК. *(Альтернатива: любой другой лаунчер, но с установкой сборки разбираться самостоятельно)*
- Java 17+ (рекомендуется использовать встроенную Java из TLauncher)  
- Достаточно свободного места на диске (не менее 1Gb) для клиентской сборки
- Не менее 4Gb оперативной памяти

## Установка сборки через TLauncher

1. **Скачать сборку из GitHub** 
	
   1.1) Первый способ (используя git):
      - Введи в командной строке `git clone -b client-new --single-branch --depth 1 https://github.com/Leg15Coder/Minecraft_2025.git`
		- `git chechout -b client-new`

	1.2) Второй способ:
	   
      - Открой репозиторий:  `https://github.com/Leg15Coder/Minecraft_2025`  
	   - Переключись на ветку `client-new`.
	   - Нажми `Code` → `Download ZIP` и распакуй архив в любую папку.

2. **Подготовить папку версии TLauncher**  
   - Открой TLauncher → вкладка **TL** → **Настройки**
   - В поле "Директория" будет указано расположения папки с майнкрафтом. Её нужно открыть.
   - Перейди в `versions/`.  
   - Создай папку `Minetest 2.0`.  

3. **Скопировать файлы сборки**  
   - Из распакованного архива GitHub скопируй все файлы и папки ветки `client`  
     (`.fabric`, `mods`, `shaderpacks`, `config`, `data`, `downloads`, `journeymap`, `natives`, `polymer`, `Minetest 2.0.jar` и т.д.)  
   - Вставь их в `.../.minecraft/versions/Minetest 2.0/`.

4. **Создать запись версии в TLauncher**  
   - В папке `.../.minecraft/versions/Minetest 2.0/` убедись, что лежит файл `Minetest 2.0.jar`.  
   - При необходимости создай `Minetest 2.0.json` и `Minetest 2.0.jar` по аналогии с любой существующей версией (скопировать и переименовать, указав правильное имя `jar`).  
   - Перезапусти TLauncher, чтобы новая версия появилась в списке.

5. **Выбор версии и запуск**  
   - В TLauncher выбери версию **`Minetest 2.0`** (или то имя, которое указано в `.json`).  
   - Убедись, что выбран правильный аккаунт и объём выделенной памяти (RAM) не менее 4Gb. В настройках TLauncher рекомендовано поставить максимальный доступный объём оперативной памяти (Изначально мог стоять слишшком маленький).
   - Нажми **Войти в игру**. 

6. **Пользовательские настройки**
  - Графические настройки могут быть выкручены на минимум. Чтобы настроить графику под себя перейдите в **Настройки** → **Настройки графики**
  - Поставьте свои настройки (прорисовка, FPS, детализация, облака и др.)
  - При необходимости можно включить/выключить или скачать новые шейдеры (используя iris)

## Как зайти на сервер

После захода в игру нажимаешь **Сетевая игра** → **Добавить** → Вводишь:
- Название сервера: Minetest2026
- Адрес сервера: 147.45.97.34:25565
- Наборы ресурсов: **Включены**
Нажимешь **Готово** → **Подключиться** → Наслаждаешься игрой!

## Проблемы и возможные решения

Если TLauncher после перезагрузки не видит новую версию можно попробовать сделать следующее:

1. Создай новую версию в TLauncher в ручную. Она должна соответствоать:
   - Minecraft 1.21.1
   - Fabric 0.18.4
   - Дополнительные настойки (галочки) выключены
   - Выделение памяти - максимально доступное
2. Запустите игру на этой версии.
3. Когда игра запустится, сразу выйдете из неё.
3. В папке `.../.minecraft/versions/` должна была появится ваша версия. Скопируйте содержимое папок `mods`, `shaderpacks`, `resourcepacks` из распакованного архива GitHub в аналогичные папки вашей версии.
4. Перезапустите TLauncher и войдите в игру

## Голосовой чат

В игре доступен голосовой чат Simple Voice Chat. Чтобы его активировать, после захода на сервер нажмите клавишу "M", а после чего будут доступны расширенные настройки по клавише "V". В поле "режим активации" лучше выбрать "активация по голосу". Прежде чем нажимать на клавиши, проверьте что нет конфликтов в *Настройки* → *Управление*. Если конфликты есть - решите их любым удобным для вас способом.

## Обновление сборки

Чтобы обновить клиент до новой версии из репозитория:

1. Скачай свежий ZIP ветки `client` или сделай `git pull`, если используешь git.
2. Замените содержимое папки `.../.minecraft/versions/Minetest 2.0/` файлами из обновлённого репозитория.  
3. При изменении имени `.jar` не забудь обновить его в `Minetest 2.0.json`.
